# БО «БФ «Безпечна Україна» — сайт

Статичний сайт-візитівка фонду. Напрям дизайну: **01 · Телеметрія** (темна приладова естетика).
Без бекенду — чистий HTML/CSS/JS, готовий до розміщення на **GitHub Pages**.

## Склад

```
index.html    — сторінка з усіма секціями
styles.css    — стилі та дизайн-токени (кольори, шрифти)
script.js     — мобільне меню, поява секцій при скролі, годинник у hero
README.md     — цей файл
```

Шрифти (Space Grotesk, Inter, IBM Plex Mono) підвантажуються з Google Fonts — потрібен інтернет у відвідувача. За потреби їх можна вшити локально пізніше.

## Як опублікувати на GitHub Pages

1. Створіть репозиторій на GitHub (наприклад `bezpechna-ukraina`).
2. Завантажте всі файли (`index.html`, `styles.css`, `script.js`, `README.md`) у корінь репозиторію — кнопка **Add file → Upload files**, потім **Commit changes**.
3. Відкрийте **Settings → Pages**.
4. У розділі **Build and deployment → Source** оберіть **Deploy from a branch**.
5. Гілка — **main**, тека — **/ (root)**. Натисніть **Save**.
6. Через 1–2 хвилини сайт буде доступний за адресою
   `https://<ваш-логін>.github.io/<назва-репозиторію>/`

### Власний домен (опційно)

У **Settings → Pages → Custom domain** впишіть домен (напр. `bezpechna-ukraina.org`)
і додайте відповідні DNS-записи у реєстратора. GitHub згенерує файл `CNAME` автоматично.

## Як редагувати текст

Увесь текст — прямо в `index.html` між тегами секцій (`<section ...>`).
Кольори й шрифти змінюються у `styles.css` у блоці `:root` згори.
